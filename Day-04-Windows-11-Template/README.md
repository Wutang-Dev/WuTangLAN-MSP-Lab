# Day 04 – Windows 11 Template

## Objective
Create a Windows 11 template that will be used to deploy workstation clients in the MSP lab.

## Template Location

D:\VM-Templates

## Template Created

Windows 11

This template will be used to create the following clients:

CLIENT-EMIRATES  
CLIENT-HIGHBURY  

Additional clients may be deployed later.

## Benefits of Templates

Using templates allows machines to be deployed quickly while maintaining consistent configurations across the environment.

## Outcome

A Windows 11 template is now available to deploy client workstations for the WuTangLAN MSP lab.

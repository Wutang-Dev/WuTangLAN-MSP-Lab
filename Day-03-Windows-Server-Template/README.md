# Day 03 – Windows Server Template

## Objective
Create a Windows Server template that will be used to deploy servers in the MSP lab.

## Template Location

Templates are stored on a secondary drive:

D:\VM-Templates

This prevents the main lab drive from being filled with template images.

## Template Created

Windows Server 2022

Purpose:
- Used to deploy DC01
- Used to deploy FS01
- Used for future infrastructure servers

## Configuration

Generation 2 VM  
Dynamic VHDX  
Minimal resources to keep the lab lightweight.

## Outcome

A reusable Windows Server template has been created to simplify future server deployments in the WuTangLAN MSP lab.
